## About Me

I'm an AI/ML Engineer who enjoys turning research ideas into working, production-ready systems. I like digging into data, experimenting with models, and shipping things that actually solve problems — not just notebooks that sit unused.

When I'm not training models, I'm usually exploring new tools in the ML ecosystem or refining the pipelines behind the scenes that make everything run smoothly.

## Areas I Work In

- 🤖 **Machine Learning & Deep Learning** — model design, training, and evaluation
- 🧠 **NLP & LLMs** — building and fine-tuning language-model-driven applications
- 👁️ **Computer Vision** — image and video-based models
- ⚙️ **MLOps** — model deployment, monitoring, and pipeline automation
- 🗄️ **Data Engineering** — data pipelines, preprocessing, and feature engineering
